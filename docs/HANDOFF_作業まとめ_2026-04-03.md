## production black-box 検証

### 確認済み

- `GET /api/app-snapshot?scope=invalid` -> `400`
- 認証済み bearer 付き `GET /api/app-snapshot?scope=home` -> `200`
- `POST /api/auth/anonymous-link/prepare` -> `200`
- 匿名連携 normal flow
  - `prepare -> complete -> same nonce retry` がすべて `200`
- 匿名連携 retry flow
  - 失敗注入後の再試行でも `200`
- `POST /api/visit-logs` -> `201`
- `SECURITY_TEST_BASE_URL=https://maguromaru-note.vercel.app npm run test:security:blackbox` -> 成功

### 重要

匿名連携の retry は、black-box で一度 `500 User not found` を再現済みだったが、`0a105f6` で修正し、production でも再確認済み。
